Una base datos es un conjunto organizado de datos que se almacena  en una estrructura facil de gestionar.
-Conceptos Clave:
  Tabla la plantilla donde se va hacer la base de datos.
  Registro es como el objeto la instacia.
  Campo es como el tipo de dato como un atributo.
  Clave Primaria es el campo unico qeu identifica cada registro  para una actualizacion segura.
  Consulta es hacerle una pregunta a la abse de datos.

Se dividen en relacional y no relacional
  -La relacional se usa para datos estructurados(Ordenado).
  - La no relacional se usa para datos no estructurados.
