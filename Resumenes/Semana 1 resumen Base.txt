RESUMEN DE LA SESION: INTRODUCCION A BASE DE DATOS

DEFINICION Una base de datos es un conjunto organizado de informacion que se almacena de forma estructurada para facilitar su acceso, gestion y actualizacion. Un ejemplo sencillo es una agenda de contactos, donde cada persona tiene nombre, telefono y correo, formando una tabla con filas y columnas. 
CONCEPTOS CLAVE
Tabla: Similar a una hoja de calculo, compuesta por columnas y filas. 
Registro: Una fila que contiene datos de una sola unidad, como un cliente o un libro. 
Campo: Una columna que representa un tipo de dato especifico, como nombre o precio. 
Clave primaria: Campo unico que identifica cada registro, por ejemplo, el DNI o un ID. 
Consulta (query): Preguntas realizadas a la base de datos para obtener informacion especifica. 
TIPOS DE BASES DE DATOS
Relacional (SQL): Estructurada mediante tablas y relaciones. Ejemplos: MySQL, PostgreSQL, SQL Server. 
No relacional (NoSQL): Flexible para datos no estructurados. Ejemplos: MongoDB, Firebase.
